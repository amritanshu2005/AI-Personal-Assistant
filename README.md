# ⚡ AI Personal Assistant

A sleek, dark-themed AI-powered web assistant that can answer questions and summarize emails instantly using high-speed LLM inference.

---

## 🧠 What it does

This project is a lightweight AI tool that provides:

* 💬 **Smart Question Answering**
* 📧 **Email Summarization (2–3 line concise output)**
* ⚡ **Fast responses powered by Groq (LLaMA 3)**
* 🎯 **Minimal, distraction-free UI with a modern AI aesthetic**

---

## 🚀 Demo Preview

> A clean, terminal-inspired interface designed for focus and speed.

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS, JavaScript
* **AI Engine:** LLaMA 3 via Groq API
* **Environment Management:** python-dotenv

## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/amritanshu2005/AI-Personal-Assistant.git
cd AI-Personal-Assistant
```

---

### 2. Install dependencies

```
pip install -r requirements.txt
```

---

### 3. Create `.env` file

Create a `.env` file in the root directory:

```
API_Key_GROQ=your_api_key_here
```

---

### 4. Run the application

```
python main.py
```

---

### 5. Open in browser

```
http://127.0.0.1:5000
```
